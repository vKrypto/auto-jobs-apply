# Frontend

Minimal React + TypeScript frontend for the Auto Apply application.

## Setup

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

The frontend will run on `http://localhost:3000` and proxy API requests to `http://localhost:5000`.

## Build

```bash
npm run build
```

## Features

- Single button to trigger auto-apply feature
- Real-time status display (pending, success, failure)
- Hardcoded preferences (keywords, job filters, application settings)
- Minimal CSS with inline styles
- Modular component structure
